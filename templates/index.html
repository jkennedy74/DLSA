Recurrent Neural Networks (RNNs)

  • Useful with sequential data because each neuron uses its internal memory to maintain information about the previous input.
  • Can be thought of as multiple copies of the same network each passign a message to its successor.
  • Common applications include speech recognition, language modeling, translation and image captioning.
  • RNNs are limited in how they handle long-term dependencies
    • Learn using past information if the gap between the relevant information and the place it is needed is small.
    • If this gap is large, RNNs become unable to learn how to connect the information

Long Short-Term Memory Networks (LSTMs)

  • LSTMs are RNNs that are capable of learning long-term dependencies.
